# Spring Framework 공부

Spring Framework 강의를 통해 게시판을 제작해보며 Spring Framework에 대한 지식을 습득하였습니다.

## 사용 기술

  - Spring MVC
  - JSP
  - Oracle Database
  - Apache Tomcat
  - MyBatis
